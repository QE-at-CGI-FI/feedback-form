# Vibe Coding Feedback App

Browser app to collect session feedback with:

- Star-based scoring (1-5)
- Improvement ideas for next time
- Local Storage persistence (data stays in your browser)

## Run

1. Open `index.html` in your browser.
2. Select a star score.
3. Add ideas for what to do better next time.
4. Click **Save Feedback**.

All entries are saved in `localStorage` under the key `vibe-feedback-entries`.
